Tieme Machine

Tieme Machine - это таймер, который поможет отслеживать ваше время, затраченное на любые действия.

Запуск таймера произойдет после нажатия на z + x

Для записи затраченного интервала времени в таблицу необходимо повторно нажать z + x, после этого действия таймер не останавливается, а продолжает отсчитывать время, но уже с текущего момента.

Копирование всех полученных данныех из сформированной таблицы в буфер обмена осуществляется нажатием o + p

Удалить все данные из таблицы d^

Дополнительно

Если вы случайно закрыли вкладку, когда таймер активен, вы не потеряете прогресс. После открытия данной вкладки повторно, таймер продолжит отсчет с значения сохраненного перед закрытия вкладки.

Таймер активен только шестнадцать часов, после Local Storage отчищается.

Все полученные результата хранятся в Local Storage.

Все предложенные комбинации клавиш на латинице.
